# eparser. The expression parser.

Small expression parser.

## a calculator example 

see tests/calculator.cpp

```
Enter: 2 + 2 * 2
        (2 + (2 * 2)) = 6
Enter: (2 + 2) * 2
        ((2 + 2) * 2) = 8
Enter: 11.11 / 45.9 * 5467 + 321 - 1024 / 234 * 43
        ((((11.11 / 45.9) * 5467) + 321) - ((1024 / 234) * 43)) = 1456.11
Enter: q

```
